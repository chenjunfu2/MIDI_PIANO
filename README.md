# MIDI_PIANO
无聊的MIDI钢琴软件，可以录制和回放</br>
</br>
使用说明： 无聊的弹琴软件，带录制回放功能</br>
按下按键会播放对应钢琴声，并且会实时写入录制文件</br>
打开软件即开始录制，关闭软件结束录制</br>
文件生成在软件当前目录下</br>
拖动生成的录制文件到软件上就会自动开始播放，结束自动退出</br>
</br>
  按键&nbsp&nbsp&nbsp音高</br>
